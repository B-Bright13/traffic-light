API Reference & Explorer

GET /api/status

Returns the currently active light color and metadata.

Response


          
{
  "active": "red",
  "updated_at": "2026-04-14T12:00:00+00:00",
  "color_info": { "label": "Red", "meaning": "Stop", "hex": "#ef4444" }
}

        

      


      
      

        

          

            

              GET
              /api/colors
            

            
Lists all valid colors with labels, meanings, and hex values.

          

          
        

        

          
Response

          

          
{
  "colors": ["red", "yellow", "green"],
  "details": { "red": { "label": "Red", "meaning": "Stop", ... }, ... }
}

        

      


      
      

        

          

            

              POST
              /api/set
            

            
Sets the active light. Broadcasts the change to all SSE clients.

            

              
              
              
            

          

        

        

          
Example request body

          
{ "color": "green" }

          

        

      


      
      

        

          

            

              GET
              /api/stream
            

            

              Server-Sent Events stream. Connect with
              EventSource to receive real-time updates.
              Sends a heartbeat comment every ~25 s to keep the connection alive.
            

          

        

        

          
Example client

          
const es = new EventSource('/api/stream');
es.onmessage = (e) => {
  const state = JSON.parse(e.data);
  console.log(state.active); // "green"
};
